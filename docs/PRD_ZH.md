# TIED 平台 - 产品需求文档

---

## 执行摘要

TIED（透明投资证据与发现平台）是一个AI驱动的估值平台，通过让任何人都能构建DCF模型、通过多Agent辩论生成假设、并参与透明的假设共识市场，实现金融分析的民主化。

---

## 问题陈述

1. **高门槛**: 传统估值需要Excel技能、金融培训和大量手工工作
2. **假设不透明**: 卖方分析师报告不展示假设背后的推理过程
3. **无假设市场**: 与股价不同，估值假设没有透明的市场
4. **信息不对称**: 机构投资者拥有比散户更好的分析工具

## 验证机制

### 可验证的假设类型

| 假设类型 | 验证方式 | 时间周期 | 是否参与排名 |
|----------|----------|----------|--------------|
| 当季假设 (Q1) | ✅ 财报发布 | 约3个月 | ✅ 是 |
| 当年假设 (FY) | ✅ 年报发布 | 约12个月 | ✅ 是 |
| 远期假设 (FY+1, +2...) | ❌ 暂无法验证 | 数年 | ❌ 否（仅展示） |

### 声誉评分公式

```
声誉分 = Σ (预测准确度 × 时间权重 × 难度系数)
```

### 假设分类

| 类型 | 假设 | 可验证？ |
|------|------|----------|
| **经营类** | Revenue, EBIT Margin, CapEx, D&A | ✅ 财报验证 |
| **估值类** | WACC, Terminal Growth, Tax Rate | ⚠️ 间接验证（见下文） |

### WACC验证方式

您说得对！WACC可以间接验证：

| 组成部分 | 验证方式 |
|----------|----------|
| Risk-free Rate | ✅ 国债收益率可观察 |
| Beta | ✅ 市场数据可计算 |
| Equity Risk Premium | ⚠️ 事后股权回报可对比 |
| Cost of Debt | ✅ 债券收益率可观察 |

**验证逻辑**：如果用户选择的WACC=9%，而实际股权回报长期是12%，说明WACC估得太保守。

## 后台模型训练

### 目的

用户生成的假设，一旦经过财报验证，就成为**高质量的训练数据**。这形成了一个飞轮效应：

```
用户提交假设 → 财报验证 → 收集准确数据 → 微调模型 → AI建议更准 → 更多用户
```

### 训练数据收集

| 数据类型 | 来源 | 价值 |
|----------|------|------|
| 假设+实际值 | 已验证的预测 | 微调的Ground Truth |
| 用户调整 | 参数修改记录 | 揭示专家直觉 |
| 置信度水平 | 质押金额 | 加权训练信号 |
| 推理文本 | 用户解释说明 | 思维链训练数据 |

### 训练策略

1. **持续学习**: 每季度用验证后的假设微调基础模型
2. **行业专用模型**: 为不同行业训练专门的模型
3. **用户个性化Agent**: 可选的基于用户历史的个性化微调

### 隐私与授权

- 所有训练数据匿名化处理
- 用户选择性参与贡献训练数据（有奖励）
- 训练集中不包含任何PII（个人身份信息）
